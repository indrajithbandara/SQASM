# Simple Quantum Assembly

SQASM is positioned as a simple low-level quantum programming language that anybody can understand.

## Installation

git clone .. 
requires LEX, YACC and Python 2.7+

## Usage

edit compile.sh to include ex.sqasm
bash compile.sh
profit
